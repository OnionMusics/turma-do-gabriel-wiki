Índice rápido

Use os atalhos abaixo para organizar o conteúdo.

  1. Personagens (páginas individuais)
  2. Histórias (cronologia / sinopses)
  3. Jogos (páginas com sprites e mecânicas)
  4. Músicas (letras e fontes dos instrumentais)
Guia rápido — criar nova página

1) Duplique um template, 2) ajuste metadados, 3) salve como personagens/nome.html

Deploy

Sugestões: GitHub Pages, Netlify ou Vercel. Basta enviar este diretório e apontar o index.html como homepage.

Contato

Crie uma página /contato.html com formulário ou apenas um e-mail para admin.
